# Procrustes-Analysis
Procrustes Rotation Analysis Tutorial R Scripts from Just One Bird's Opinion YouTube video. Includes information for a tutorial on comparing sites and for comparing species. 

If you are interested in using a Procrustes Analysis to compare sites - check out this video: https://www.youtube.com/watch?v=bUuWFO3wH5s&t=4s
If you are interested in using a Procrustes Analysis to compare species/variables - check out this video: https://www.youtube.com/watch?v=uCBYNtux_FU&t=1s 

Comment on the respective video if you have any questions! Happy coding!
